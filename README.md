# Diplom_2

##  [Тестируемое приложение](https://stellarburgers.nomoreparties.site/)
## Используемые библиотеки:
- Java 11
- Maven 3.8.6
- JUnit 4
- Selenium 4
- Allure
- RestAssured

## Команды:
- для запуска тестовых сценариев
  ```mvn clean test```
- для генерации отчета ```allure serve target/allure-results/```
